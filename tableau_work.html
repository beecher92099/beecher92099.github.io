Showcase of Tableau Work

A brief showcase of some dashboards I've been with Tableau.

Bookings Performance Summary

This dashboard shows the bookings per selected time period (e.g., year, quarter, month, etc.). Filters are provided for different dimensions in the data. Absolute bookings are shown along with YoY change.

Active Subscriptions

This dashboard shows information about Active Subscriptions such as the number of subscriptions, number of distinct customers and partners, and annual and total contract values. Different breakdowns as provided such as number of subscriptions including SaaS only products.
