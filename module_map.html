Full Stack Web Application for Factory Module Map

A full-stack web application to allow displaying the location of modules on the factory floor and surrounding site property. Read the inventory of modules for a given project from the Azure SQL database repository. Support 4 location areas for modules: 1) Available, 2) Site Image Area, 3) Stored Offsite, and 4) Delivered. Support the ability to drag modules between the 4 locations. On the Site Image Area support the ability to drag modules to specific locations and rotate modules in 45 degree increments. Support selecting multiple modules at once and dragging as a group. Support alignment operations on a selected group: left or top alignment, vertical or horizontal equal distribution. Support the ability to search for a specific module and highlight where the module is located. Support password protected edit mode to enable module location editing. Ability to view module locations in the past (time travel mode). Ability to undo/redo edits. Support desktop browser with mouse operations along with tablet/mobile devices with touchscreen operations. Coding was done using Claude Code.

Backend

  • Key Features: Graceful shutdown, health checks, Azure-optimized deployment

Frontend

  • Framework: React 19 with TypeScript
  • Build Tool: Vite
  • State Management: TanStack React Query for server state
  • UI Libraries:
    • Konva.js for canvas-based map interactions
    • DnD Kit for drag & drop
    • Tailwind CSS for styling
  • Architecture: Component-based with hooks for state logic
